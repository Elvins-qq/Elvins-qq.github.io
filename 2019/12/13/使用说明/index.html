更改网站名称

​ 修改Elvins目录下的_config.yml文件，修改title后的内容，然后执行提交步骤即可(见写/修改文章下的提交三部曲)

更改网站主页头像

​ 把想要更换的图片改名为avatar.png，然后放到Elvins\themes\nexmoe\source\image 目录下(覆盖原图片)，然后执行提交步骤即可(见写/修改文章下的提交三部曲)

更改”关于我”主页的内容

修改关于我的背景图

​ 把想要的图片改名为about.jpg，然后放到/Elvins/source/img目录下(覆盖原来的图片)，然后执行提交步骤即可(见写/修改文章下的提交三部曲)

修改关于我的简介

​ 打开Elvins/source/下的”关于我.md”，然后把 <! – more –> 下面的内容替换为你想要展示的简介内容即可，然后执行提交步骤即可(见写/修改文章下的提交三部曲)

关于内容

​ 文章的标题、列表、表格之类的，可以看一下编辑菜单、段落菜单、格式菜单进行对应设置，学起来很简单。抛出个学习教程：

​ https://www.jianshu.com/p/a6a6a22e9393

关于图片引入

​ 把文章内部引入的图片放到Elvins\source\_posts\中对应的文章文件夹下，然后再文章md文件中通过格式 ![](文章目录/图片名.后缀) 来引入，或者直接把图片拖拽到.md文件中(注意如果是拖拽的话，也要改成文章目录/图片名.后缀的形式)

​ 注意图像大小最好不要超过40kb，否则加载时间过长

​ 推荐图像网站：https://www.woyaogexing.com/touxiang/

提交

​ 编辑完文章保存后，在终端（也就是右键文件夹elvins，然后选择 Git Bash Here 打开的终端）

​ 执行 hexo clean 清除缓存

​ 然后终端执行 hexo g 生成静态文件

​ 最后终端执行 hexo d 提交到网站
